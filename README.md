# Interview Assistant

Electron + React app for real-time interview assistance:
- live transcription (Deepgram)
- AI response suggestions (OpenAI-compatible API)
- knowledge base (text/images/PDF)

## Quick Start

```bash
pnpm install
pnpm start
```

That is all.  
`pnpm install` already handles Electron setup automatically.

## App Setup (first run)

1. Open **Settings**.
2. Add your **OpenAI API key** and **Deepgram API key**.
3. (Optional) Set custom API base/model.

## Build

```bash
pnpm make
```

## Tech Stack

- Electron Forge
- React + TypeScript
- TailwindCSS + DaisyUI
